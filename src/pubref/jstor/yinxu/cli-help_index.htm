1. List Of All Commands

No. Cmds/Verbs Options* Type** Descriptions Samples Notes
1 cd generic change target's path cd /system/ enter system sub target
2 create strict create a new child target create ./account/users/Fred password=123 create a user Fred
3 delete strict delete a created target delete ./account/users/Fred delete a user from account
4 exit generic exit cli mode
5 set strict change targets's one or more properties set ./account/users/Fred timeout=900 change user's property
6 show -9p --filter generic display target's properties and children targets. show ./system -3p show 3 depth sub-target w/ all properties
7 help generic cli help
8 start strict start a targets
9 stop strict stop a target
10 version generic Cli version infor and Trilla version info
11 reboot generic reboot appliance
12 shutdown generic shutdown appliance
* Notes For Options:
  • -9 : show sub-targets depth ( 0 -- 9 ) .
  • -p : show properties of all sub-targets
  • -m : show page by page, like linux 'more' cmd, SPACE: next page; ENTER: next line; q: quit.

Example:

1) show
2) show -1
3) show -2
4) show -1p
5) show -2p

show also supports long option --filter or --hide.

  • --filter property=val : display items that meet property==val condition.
  • --hide property=val: hide items that meet property==val condition.

Example:

1) show system/admin/account/users --filter username=admin
2) show system/admin/account/users --hide username=admin


** Notes For Type:
    generic : cmds that apply to all targets.
    strict : cmds that only apply to some specific targets. see Map Of Strict Applied Commands And Targets

6. Appendix-2 : Script File Usage For CLI

To use local script file for cli, the local script file must have option '-fu' for cli. Sample of cli-script file(filename:a_sample) is like following:

#!/usr/bin/cli -fu
# To enable debug, use option -dfu.
#
# Notes:
# - do not power off during test.
# - order matters.
# - bad testing design may cause appliace in unrecoverale state.
#



show # show default.
cd /system/ # change current target.
pwd # display current target path.
show # show targets after path is changed.
cd / # go back to root target.
show # show targets in root.

exit # after exit, no any cmds will be executed.

show # no operation after exit.
mbtest UI.GetUser username:admin # for internal advanced cli mbtest only.

#### end of the sample
